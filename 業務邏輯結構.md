# 業務邏輯結構
## 單字 Vocabulary
- 一個個單字為一張張的單字閃卡
- 配合題目會改變長相 (只顯示特定欄位+標記特定欄位為題目)
## 題目 Question
- 由單字卡 + 題型組成
## 任務 Quest
- 為老師指派功課/考試的基本單位
- 最多包含6道題目(Questions)
- 可以混搭不同題型
- 分為老師任務與機構任務
  - 老師任務 (in Note)
    - 出自帶班老師的題庫，for 特定班級
  - 機構任務 (in Course Review)
    - 出自機構管理員的題庫，for 全機構學生共用
## 課程結構
- Chatper (Chatper 1, 2, 3)
  - Unit (Unit 1, 2, 3)
    - Lesson (Lesson 1, 2, 3)
      - Quest
